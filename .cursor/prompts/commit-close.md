# Commit and Close Issue

If all the requirements of this issue are completed, commit all the changes and close the issue.

**Commit format:** See `commit-format.md` for detailed commit message format.

Commit message format should be like this:

"Commit message title.

Closes #<issue number>

Functionality completed in this issue."

**Next steps:** After committing, use `push-branch.md` to push the code, then `cleanup.md` to clean up the branch.
